   Reference to a listener's port just created.  This implements IConnectable with a default port (the port that an ELB listener was just created on) for a given security group so that it can be conveniently used just like any Connectable. E.g:      const listener = elb.addListener(...);      listener.connections.allowDefaultPortFromAnyIPv4();     // or     instance.connections.allowToDefaultPort(listener);
